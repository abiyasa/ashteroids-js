Ashteroids

v0.1.0

by Abiyasa

A simple Asteroids game using Ash Framework, HTML5 Canvas, CreateJS, and three.js

Ash Framework, originally written for ActionScript 3, is an entity-component system for game development.

It's a way to organise and manage your game code without sacrificing the game performance.

For more information about Ash Framework, including introductory articles, checkout its official page at http://www.ashframework.org/.
